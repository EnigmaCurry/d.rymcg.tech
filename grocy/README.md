# Grocy

[Grocy](https://grocy.info/)
is a web-based self-hosted groceries & household management solution for
your home.

## Configure

Run:

```
make config
```

## Install

Run:

```
make install
```

## Open in your web browser

Run:

```
make open
```

Login with the default credentials:

 * Username: `admin`
 * Password: `admin`

(You should immediately change these upon login)
